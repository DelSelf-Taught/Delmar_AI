# 🧠 Delmar_AI - Portfólio de Estudos em IA e Programação

Olá! Bem-vindo ao meu espaço de estudos e experimentos. Este repositório é uma coleção dos meus primeiros passos e projetos desenvolvidos enquanto aprendo sobre Inteligência Artificial, Ciência de Dados e desenvolvimento em geral. Aqui você encontrará desde scripts simples até implementações de modelos básicos, todos criados para consolidar conceitos e compartilhar minha jornada de aprendizado.

## 🎯 Objetivo do Repositório

O principal objetivo é documentar minha evolução, servindo como:
- Um **portfólio pessoal** dos meus projetos iniciais.
- Uma **base de consulta** para conceitos e implementações que estudei.
- Um recurso para **quem também está começando** e quer ver exemplos práticos e comentados.

## 🗂️ Projetos Incluídos (Exemplos)

Aqui estão alguns tipos de projeto que você pode encontrar (substitua pelos seus):

*   `01-analise-dados-vendas/`: Um script em Python para analisar um conjunto de dados de vendas, gerando gráficos e estatísticas descritivas.
*   `02-classificador-iris/`: Implementação de um classificador com KNN para o dataset Iris, incluindo avaliação de acurácia.
*   `03-previsao-precos-casas/`: Um modelo de regressão linear simples para prever preços de imóveis com base em features como tamanho e número de quartos.
*   `04-chatbot-simples/`: Um chatbot de regras básico usando processamento de linguagem natural com `nltk`.
*   `05-reconhecimento-imagens/`: Um experimento inicial com redes neurais convolucionais (CNNs) usando TensorFlow/Keras para classificar dígitos manuscritos (MNIST).

*(Cada projeto está em sua própria pasta, com um README específico e o código-fonte comentado.)*
